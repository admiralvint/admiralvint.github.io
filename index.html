Kids Screentime Control
Infrastructure

Parental control system for enforcing screen time limits on kids' Bazzite Linux laptops, managed from a central web dashboard. A Python daemon runs on each laptop as root, detects active Wayland sessions, accumulates usage in local SQLite, enforces daily limits and time-window rules, and reads browser history from Firefox and Brave (both Flatpaks with non-standard paths). Rules are fetched from and usage synced to the server every 15 seconds.

The FastAPI server runs in a Proxmox LXC reachable over Tailscale. The dashboard shows per-device usage bars, time windows, and recently visited sites with real-time updates — plus a force-logout that attempts a clean KDE Plasma dbus session shutdown before falling back to SIGTERM.
